Peg Wheel Launch

Peg Wheel Launch is a game I made in a very short amount of time and is still very much in development. It’s currently got the core mechanics; however, I would like to add more variations in the types of peg wheels, nicer graphics, sound design, and possible smoother gameplay.

The gameplay consists of the player launching them selves from peg to peg while the spins on wheels. There is a timer which counts down to 0 if that happens the game ends. Although, sticking a landing on a peg will increase the amount of time remaining. The goal of the game is to get as far as possible before the time runs out.

The game is OUT NOW on the Google Play Store!

  • Unity, C#

Paint Jumperino

I've always been a fan of the mobile game Geometry Dash, so I decided to make a similar-ish game that puts a spin on the formula. You can double jump, there is a lot more vertical acceleration, and it is possible to die from falling from to high up unless you double jump before hitting the ground.

The game currently features a level selector and a character selector which saves progress and the users choice of character. And of course, the game it’s self.
